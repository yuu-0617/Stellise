# Stellise (ステライズ)

毎朝の「あと5分･･･」からの遅刻をなくす、完全プライバシー重視のAIスケジュール＆二度寝防止アプリです。
寝坊、電車の遅延、急な天候の変化など、現代人の予測不可能な朝のルーティン崩れをテクノロジーの力で解決します。

## 概要 (Overview)

既存の便利なAIアプリに存在する「個人データが学習に使用されるリスク」や「煩わしい広告による行動追跡」の壁を排除した、真に安心して日常使いできるアシスタントとして開発しました。
睡眠状態・天気・交通状況をシステムが総合的に判断し、毎朝「最適なスケジュール」を全自動で提案・再計算します。

## 主な機能 (Core Features)

* **オンデバイスAIによる高精度な睡眠解析**
  * TensorFlow Liteを用いた端末内AI処理を実装。外部サーバーにデータを送ることなく、「体動」や「音声」から睡眠深度をリアルタイム解析し、眠りの浅いタイミングでアラームを起動します。
* **外部データ連携による「全自動ルーティン再計算」**
  * 交通APIと天気APIを常時監視し、電車の遅延や気温を検知。遅延発生時には、出発時間に間に合うよう朝のタスク配分をAIが全自動で短縮・再構成します。
* **強力な二度寝防止アラーム**
  * 計算問題で脳をスイッチONにする機能や、指定のタスクをこなさないと止まらない仕組みで寝坊をブロックします。
* **睡眠スコアと生活リズムの可視化**
  * 毎日のリズムを記録し、睡眠をスコア化して分かりやすく提示します。

##  使用技術 (Tech Stack)

* **Language/Framework:** Swift, SwiftUI
* **Machine Learning:** TensorFlow Lite (オンデバイス処理)
* **API Integration:** 天気API, 交通API

## プライバシー・バイ・デザイン (Privacy by Design)

本アプリは、情報のローカル完結を徹底しています。
* **広告の完全排除:** アプリ内広告を一切排除しています。
* **データの非中央集権化:** 個人データや音声解析を外部クラウドに送信せず、全て端末内の隔離された環境で処理することで物理的に漏洩リスクを遮断しています。
* **学習データの保護:** ユーザーの個人データをAIの学習に二次利用させない、極めてクリーンで安全なパーソナライズ環境を構築しています。ユーザーのフィードバックも個人の端末内にのみ安全に保存されます。
